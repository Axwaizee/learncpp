C++ Beginner Questions

1. Basic Input / Output

  1. Write a program that prints Hello, World! to the screen.
  2. Ask the user for their name and print a greeting with their name.
  3. Ask for two numbers and print their sum.
  4. Ask for your age and print how old you’ll be next year.

2. Variables and Data Types

  1. Declare variables of different types and print their values.
  2. Convert minutes into seconds.
  3. Ask the user for the radius of a circle and print the area.

3. Conditions (if / else)

  1. Check if a number is positive, negative, or zero.
  2. Ask the user for their age and print whether they are an adult.
  3. Check if a number is even or odd.
  4. Ask for a test score and print if the student passed or failed.
  5. Ask for two numbers and print the larger one.

4. Loops

  1. Print numbers from 1 to 10 using a for loop.
  2. Print the sum of numbers from 1 to 100.
  3. Ask for a number n and print all numbers from 1 to n.
  4. Use a while loop to keep asking the user for a number until they enter 0.
  5. Print the multiplication table of a number entered by the user.

5. Arrays and Loops

  1. Ask the user to enter 5 numbers, store them in an array, and print them back.
  2. Find the largest number in an array of 5 integers.
  3. Calculate the average of 5 numbers entered by the user.

6. Functions

  1. Create a function that adds two numbers and returns the sum.
  2. Create a function that takes a number and returns whether it’s even or odd.
  3. Create a function that prints a line of stars based on a number given by the user.

7. Strings

  1. Ask the user for their full name and print the number of characters in it.
  2. Ask the user for a word and print it in reverse.
  3. Ask for two words and print them combined with a space in between.

8. Simple Practice Projects

  1. Make a simple calculator that can add, subtract, multiply, or divide two numbers.
  2. Ask for a password and print "Access granted" only if it matches a fixed password.
  3. Write a guessing game: the program has a fixed number and the user tries to guess it.
  4. Create a program that prints a triangle pattern of stars.
